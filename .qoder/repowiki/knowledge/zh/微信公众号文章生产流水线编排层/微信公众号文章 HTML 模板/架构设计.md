本模块仅包含静态 HTML 模板文件，无运行时代码，由上层 Python 脚本（如 step2_table_to_html.py、step3_json_to_html.py）通过字符串替换注入内容：
- caicai_html_1_green_classical.html：经典长文模板，内嵌全局 CSS（标题/正文/高亮/空行等 class），在 `<article id="clipboard-content">` 中预留 `{{BODY_PLACEHOLDER}}` 供 JSON→HTML 渲染结果填充；头部与底部分隔线、推荐阅读等装饰区固定不变。
- caicai_html_1_green_table.html：表格专用模板，绿色表头 + 斑马纹行，预留 `{{TABLE_PLACEHOLDER}}` 供 table_to_html 生成器插入 `<table>`，并附带一段 onload 脚本将 tbody 各 tr 行高统一为最高行，以适配微信编辑器对表格行高的渲染差异。
- caicai_html_1_green_classical_instance.html：已渲染的实例输出，作为人工校对参考。
依赖方向单向：上层生成器 → 模板占位符替换 → 产出可直接粘贴至微信公众号后台的完整 HTML。
