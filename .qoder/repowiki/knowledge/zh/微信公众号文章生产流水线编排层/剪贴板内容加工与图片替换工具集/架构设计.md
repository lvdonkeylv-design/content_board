本模块由两个独立可执行的 Python 脚本组成，均直接通过 ctypes 调用 user32/kernel32 的 Win32 剪贴板 API（OpenClipboard/GetClipboardData/SetClipboardData/GlobalAlloc 等），不依赖第三方库。
- `tool_clipboard_maimai.py`：纯文本流水线——读取剪贴板 CF_UNICODETEXT → 截断前 600 字 → 解析同级 `wx_content_list.txt`（标题/空行/URL 四行一组）→ 按「展开内容如下」+「其他优质内容分享」两段拼接 → 写回剪贴板。
- `tool_clipboard_replace_images.py`：HTML 流水线——注册并读取「HTML Format」格式（解析 StartFragment/EndFragment 字节偏移提取 fragment）→ 读取 `content_instance/<实例>/process/step2_table_to_image.json` 获取图片顺序 → 用正则匹配 `<img src>` 并从后向前替换为本地图片的 data URI → 执行 postprocess_html（去除 background-color、margin 归零、清理冗余 span）→ 追加 `add_html.html` 尾部声明 → 同时写入 HTML Format + CF_UNICODETEXT + CF_LOCALE(2052) 三种格式。
- 数据文件：`wx_content_list.txt` 提供微信文章条目；`add_html.html` 作为 HTML 尾部模板被后者动态加载。两脚本彼此无依赖，各自以 `if __name__ == "__main__"` 入口运行。
