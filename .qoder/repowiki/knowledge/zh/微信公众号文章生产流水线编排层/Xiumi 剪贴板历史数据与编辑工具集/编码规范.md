- 所有 CLI 脚本采用 `if __name__ == "__main__": sys.argv` 模式，并在缺少参数时打印 Usage 后 `sys.exit(1)`。
- 二进制写入统一通过内部 `save(fmt_id, fmt_name, data)` 闭包完成，文件名按 `{fmt_id:05d}_{safe_name}.bin` 命名并追加到 manifest 列表，避免分散的文件 IO 逻辑。
- HTML 片段识别/折叠使用一组顺序执行的 `re.sub` 正则（title → body-bold → body → empty-line → inline highlight），并通过 `(?:(?!</p>).)*` 等负向前瞻防止跨标签匹配。
- 日志输出统一以 `[INFO]` / `[WARN]` / `[ERROR]` / `[DONE]` 前缀打印到 stdout，便于管道化处理。
- 所有 JSON 序列化使用 `indent=2, ensure_ascii=False`，保证人类可读且保留中文。
